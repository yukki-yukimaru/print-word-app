# 仕様書：プリント英単語 発音アプリ

- **アプリ名**：プリント英単語 発音アプリ
- **作成日**：2026-04-22
- **バージョン**：1.1.0
- **公開URL**：https://yukki-yukimaru.github.io/print-word-app
- **リポジトリ**：https://github.com/yukki-yukimaru/print-word-app

---

## 実装機能一覧

| 機能 | 内容 |
|---|---|
| 画像アップロード | ファイル選択 / カメラ撮影 / ドラッグ＆ドロップ |
| AI読み取り | Gemini APIで英単語・フレーズ・長文を抽出 |
| 発音 | 単語テキストまたは🔊ボタンをタップでブラウザ音声読み上げ（英語・0.9倍速） |
| 編集 | ✏️ボタンでインライン編集。Enterで確定、Escでキャンセル |
| 削除 | 🗑️ボタンで単語カードを個別削除。語数を自動更新 |
| 長文対応 | 例文・長い文章もそのまま抽出・発音できる |
| APIキー管理 | LocalStorageに保存（ページ外部への送信なし） |
| エラー表示 | 混雑中・タイムアウト（30秒）・API不正などを日本語で表示 |

---

## 使用技術

- **言語**：HTML / CSS / JavaScript（シングルファイル構成）
- **AI**：Google Gemini API（`gemini-3-flash-preview`）
- **音声**：Web Speech API（SpeechSynthesisUtterance）
- **ホスティング**：GitHub Pages

## 起動方法

ブラウザで以下のURLを開く：
```
https://yukki-yukimaru.github.io/print-word-app
```

ローカルで確認する場合は `index.html` をブラウザで直接開く。

---

## 操作手順

1. Gemini APIキーを入力して「保存」
2. 英単語プリントの画像を選ぶ（ファイル選択 or カメラ or ドラッグ）
3. AIが英単語・フレーズ・例文を自動抽出
4. カードをタップ or 🔊で発音
5. 読み取りミスは ✏️ で編集、不要なカードは 🗑️ で削除

---

## 既知の問題・注意事項

- Gemini APIが混雑している時間帯（夜など）は「AIが混雑中」エラーが出ることがある → 時間をおいて再試行
- APIキーはGoogleAI Studioで取得（無料枠あり）
- 画像が大きすぎると読み取りに時間がかかる場合がある
